# 💕 Propuesta Especial para Yaneli

Una invitación romántica e interactiva creada con HTML, CSS y JavaScript.

## 🌟 Características

- Diseño glassmorphism con gradientes animados
- Elementos flotantes en el fondo

- Celebración con confetti al aceptar
- Control de música de fondo
- Redirección automática a WhatsApp
